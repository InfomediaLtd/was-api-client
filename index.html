Login


In order to use the Infomedia API you should first log in. You can use your username and password for Infomedia's apps. If you've been given a token by Infomedia then you can use that for logging in.

The Login service will generate an Access Token that grants you access to all authorized Infomedia API services. You should pass it in the "Authorization" header in all API calls.

You can view the full documentation of the Authentication service here:
http://api-sandbox.superservice.com/v1/auth/docs/

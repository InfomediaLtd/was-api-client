Login


In order to use the Infomedia API you should first log in. You can user your username and password. If you've been given a Refresh Token by Infomedia then you can use that for logging in.

The Login service will generate an Access Token that should be used to access all other Infomedia API services.

You can view the full documentation of the Authentication service here:
http://api-sandbox.superservice.com/v1/auth/docs/
